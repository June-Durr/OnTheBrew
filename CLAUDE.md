# On The Brew Coffee Company — SiteLaunch Studios Build

## Project

Custom marketing site for On The Brew Coffee Company (onthebrew.com).
Guatemalan heritage coffee roaster, founders Cecilia and Jonathan.
Physical shop: Stamford Town Center, 100 Greyrock Place, Level 3 Green South.

## Stack

React + Vite + Tailwind CSS + Netlify
Animation: motion package (npm: framer-motion). MIT license. Zero client cost.
No other animation libraries unless explicitly approved.
No localStorage. No browser storage APIs.

## Routing

Brand and visual decisions → otb-brand.md
Architecture, routing rules, build order → otb-architecture.md

## Prompting rules for this project

Always ask for one section at a time.
Name specific references (Haven Coffee, Melnudens) when requesting styling.
Request motion package explicitly by import name when adding animation.
Keep components in /src/components/, one file per section.
